Decorations

Motivation
Subtle visual elements can be used to improve the readability and amount of information conveyed to a viewer.
Philosophy
Decorative features of a document should be used to convey useful information, not just to make a site look more interesting.

Badges

Description

A badge is a small icon that can be used as a subtle indicator of the relevance of information.

Examples
HAML 
%i.fa.fa-info.badge.info
%i.fa.fa-check.badge.good
%i.fa.fa-exclamation.badge.warn
%i.fa.fa-bolt.badge.error

Labels

Description

A label can be used as a prominent status indicator, especially for AJAX status messages.

Examples
Info
Good
Warn
Error
HAML 
.label.info Info
.label.good Good
.label.warn Warn
.label.error Error

Progress

Description

Progress bars are indicators of completion that scale very nicely in responsive layouts.

Examples
HAML 
%progress{max:100,value:75}
%progress{max:100,value:50}
%progress{max:100,value:25}
%progress{max:100,value:60}
