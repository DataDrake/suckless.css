Helpers

Motivation
Sometimes a designer needs some assistance making individual elements look better. Helpers may be applied to any element in order to style it as desired.
Philosophy
Helpers should be applied liberally throughout a document in order to reduce the boilerplate CSS for a site. They should also be applicable to any element, not a specific type.

Alignment

Description

Alignment helpers allow objects to be easily positioned on a page.

.center

Box Model

Description

Manipulation of the box model is key to making clean looking sites. These helpers allow a designer to control the size, padding, and border of any element at will.

Typography

Motivation
Typography and type-setting can be the most complicated parts of a new design. Default styles and type helpers are essential to a developer.
Philosophy
Sane defaults for type-setting of headers and inline text allow a designer to focus less on font-sizes and decoration and more on the alignment, position, and color of type.

Headers

Description

Header sizes and decoration are given sane defaults requiring a designer to only worry about the header level and color of specific text.

Description

Text Alignment

Description

It is frequently necessary to alter the alignment of text within a container, both to make better use of the space and be visually different than nearby text. The provided text-alignment helpers may be easily applied anywhere in a document.
