Introduction

Compatibility

Browser support was been limited to: Google Chrome, Mozilla Firefox, Safari, and Microsoft Edge. As most mobile browsers are built using WebKit or Blink as a base, you can expect them to function appropriately. There are no plans to ever support older browser engines as significant portions of this library depend on newer CSS3 features like Flexbox.

This library does not depend on JavaScript for its functionality in any way and may therefore be use in environment the either require JS to be disabled or that lack a JS engine.

Organization

Suckless.css has been broken up into several categories of modules:

Base
This category contains all of the global classes that are extended in other places, as well as resets for client stylesheets and color classes which may be used for theming elements.
Decoration
Decorations are stylized elements which may be used as "eye-candy" in web pages. These elements are typically used to convey information to users, in a manner that is visually unobtrusive.
Helpers
The classes defined in this category may be used to aid in the general manipulation of the position, alignment, and box-model of any element in a document.
Input
Style classes have been provided for creating button links and for creating forms which are visually consistent across the major browsers.
Navigation
The navigation modules provide functional elements that allow users to traverse the contents of both a single page and the links to others.
Structure
Three different layouts have been provided to allow for responsive scaffolding of page contents.
Themes
A number of themes have been provided which allow Suckless.css to be immediately put into use, without the need to develop a color scheme for a new site. They also serve as examples should you decide to create custom themes.
Visuals
This module contains elements which may be used inside of a document to provide uniform and visually compelling displays of information.
