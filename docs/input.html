Buttons

Motivation
Buttons are essential input sources that allow a user to control navigation, transmit data, or interact with dynamic page content.
Philosophy
If it looks like a button, it is a button and must act like a button.

Forms

Motivation
Forms are one of the oldest ways for user to send information to a web server. They are useful for providing paper-like data collection.
Philosophy
Forms should have a similar appearance, even on different browsers. Selected form elements should visual indicate that they are selected.
